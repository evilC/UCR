GuiControl

UMLClass
UCR :: Class Diagrams :: GuiControl

Description

A GuiControl that has persistent state and Plugin-Author defined callbacks for change events (ie End User changed contents of GuiControl). May or may not have to derive this class to support some kinds of more complex GuiControl (eg ListViews).

Attributes

Visibility Name Type Description
public ParentPlugin Plugin

A reference to the Plugin class that contains this GuiControl
public Name String

Plugin-Author defined name for the GuiControl. Should be unique among other GuiControls in this plugin.
public Value String

The contents of the GuiControl.
public ChangeValueCallback BoundFunc

The Plugin-Author defined routine in the plugin to call then the GuiControl changes value.
