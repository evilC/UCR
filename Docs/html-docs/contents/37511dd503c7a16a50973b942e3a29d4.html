Profile

UMLClass
UCR :: Class Diagram :: Profile

Description

Each profile is an instance of this class. It has it's own GUI, and can contain 0 or more plugins. Profiles are a way for End-Users to organize plugins into meaningful groups. In the final product, Profiles will be able to be linked to applications, such that UCR switches to a profile if it's associated application becomes active.
