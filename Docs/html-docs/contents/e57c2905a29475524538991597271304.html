Hotkey

UMLClass
UCR :: Class Diagrams :: Hotkey

Description

A GuiControl that has persistent state and Plugin-Author defined callbacks for change events (ie End User changed contents of GuiControl). Additionally, has Plugin-Author defined callbacks for up and down events of the End User defined hotkey.

Attributes

Visibility Name Type Description
public ParentPlugin Plugin

A reference to the Parent Plugin that this Hotkey is part of.
public Name String

Plugin-Author defined name for the Hotkey. Should be unique among other Hotkeys in this plugin.
public Value BindObject

Holds the End User's current choice of hotkey
public ChangeValueCallback BoundFunc

The Plugin-Author defined routine in the plugin to call then the Hotkey changes value (eg the End-User chose another hotkey).
public ChangeStateCallback BoundFunc

The Plugin-Author defined routine in the plugin to call then the Hotkey changes state (eg the End-User defined hotkey went down or up).
